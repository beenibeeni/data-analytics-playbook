# Data Analytics Notes

Personal reference guides and study notes on SQL, Power Query, and other data analyst tools — written as I learn them.

Each guide breaks a topic down into the core concepts, the exact syntax, what the output looks like, and the mistakes to watch for. They're meant to work both as my own quick-reference and as a record of how I think through and explain technical concepts.

## Why this exists

Most of my project repos (like [ph-job-market-analytics](https://github.com/beenibeeni/ph-job-market-analytics)) focus on the finished dashboard. This repo is the opposite — it's the working-through-it part: the concepts and syntax I had to actually sit down and learn properly in order to build those projects.

## Format

Each guide is a single self-contained HTML file, published via GitHub Pages, styled the same way across topics:

- Sidebar navigation by section/module
- Numbered, expandable steps
- Copy-able code blocks
- Tip / warning / note callouts for the details that are easy to miss

## How to use this repo

- Browse the rendered guides via the links above, or
- Open any `.html` file directly if you've cloned the repo
